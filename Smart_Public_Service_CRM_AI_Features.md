# Smart Public Service CRM (PS-CRM)

## Advanced AI-Based Grievance Management Features

------------------------------------------------------------------------

## 1. AI-Based Priority Scoring System

The system assigns a **dynamic priority score** to each complaint using
Artificial Intelligence based on the following factors:

### Location Sensitivity

-   Higher priority if complaint is near:
    -   Hospitals
    -   Schools
    -   Government buildings
    -   High public traffic zones

### Number of Similar Complaints

-   If multiple complaints are reported for the same issue/location,
    priority automatically increases.
-   Helps identify mass-impact civic problems.

This ensures critical issues are addressed faster and resources are
allocated efficiently.

------------------------------------------------------------------------

## 2. AI-Based Duplicate Complaint Detection

The system automatically detects duplicate complaints using:

-   Text similarity analysis
-   Location matching
-   Time proximity analysis

### Features

-   Automatically merges duplicate complaints.
-   Increases the overall priority score when duplicates are detected.
-   Prevents clutter and reduces administrative workload.
-   Maintains a unified complaint thread for better tracking.

------------------------------------------------------------------------

## 3. Nearby Citizen Notification System

When a complaint is posted:

-   Users in nearby locations receive a real-time notification.
-   Nearby citizens can:
    -   Support the complaint
    -   Add their views
    -   Confirm the issue
    -   Provide additional evidence

### Benefits

-   Encourages community participation.
-   Validates authenticity of complaints.
-   Helps authorities understand ground reality faster.
-   Improves transparency and civic engagement.
